## Создания репозитория 

Создания репозиторя 
```sh
git init 
git add (название папки)
```
* Первой командой мы указываем,что именно этой папкой мы будем пользоваться,и git создаст в нашей папке свою скрытую папку.
* Второй командой ьы добавляем фаил.
