## Списки 

Чтобы добавить не нумерованные списки необходимо выделить пункты звездочкой (*).
Например, вот так:
* Элемент 1

* Элемент 2

Чтобы добавить нумерованный список, необходимо простопронумеровать.
Например, вот так:

1 Элемент

2 Элемент
